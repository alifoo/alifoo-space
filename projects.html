Projects

breakcore-cli

A terminal-based music player built with Go. You can choose from thousands of playlists of breakcore from different platforms and listen to it from your CLI to minimize distractions. Under construction.

Wordcaster

A raycast extension to quickly translate words or sentences using the Linguee API. Several languages supported. Built with Typescript. Under construction.

Vibe Coding Blocker

A chrome extension to prevent Vibe Coding. The extension blocks actions like copying and pasting code to AI tools or making small prompts. Built with HTML, CSS and Javascript. Under construction.

keephacking-cli

A cli-based productivity tool that helps focus and improve consistency. Still under construction.
